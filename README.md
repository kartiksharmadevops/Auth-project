# Full Stack Auth Project

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## How to Run

### 1. Clone the repo
git clone https://github.com/your-username/auth-project

### 2. Backend setup
cd server
npm install
node index.js

### 3. Frontend setup
cd client
npm install
npm start

## Features
- Signup
- Login
- JWT Authentication
- Protected Routes
